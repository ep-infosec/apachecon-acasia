微服务架构越发流行，API 作为基础设施，数量呈现爆发增长。应用与应用之间的访问关系越发复杂，外加每个微服务的快速上下线、扩缩容等，我们必须要很好解决 API 的动态管理、安全策略等新问题。

本次分享将给大家剖析这些问题的由来，以及如何用云原生 API 网关（Apache APISIX） 来彻底解决这类问题，让我们的微服务架构不仅满足当下，也能够顺利过渡到云原生为将来铺路。
